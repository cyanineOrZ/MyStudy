http 的中文翻译为 超文本传输协议 望文生义， http就是一个传输协议 其涵盖的范围不止文本文件。 HTTP协议定义Web客户端如何从Web服务器请求Web页面，以及服务器如何把Web页面传送给客户端。HTTP协议采用了请求/响应模型。 客户端向服务器发送一个请求报文，请求报文包含请求的方法、URL、协议版本、请求头部和请求数据。 服务器以一个状态行作为响应，响应的内容包括协议的版本、成功或者错误代码、服务器信息、响应头部和响应数据。 两种最常见的http请求方法 get 和 post 也是前后端的 链接的 最常用的方法 一、post post - 向指定的资源 提交 要被处理的 数据 二、get get - 从指定资源 请求 数据
