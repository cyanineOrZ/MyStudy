短路运算（逻辑中断）时逻辑运算中特殊的一类 逻辑运算符比较特殊 当使用布尔值进行运算时， 其返回值也是布尔值true和false ⭐当使用值包括字符型与数字型进行运算时，其返回值为值 数字型（多用在短路运算） { 在值运算时， 除了0 ''（空字符串） null undefin NaN 其余都为真 } 短路运算原理：当有多个表达式时（逻辑运算一定是多个表达式），左边的表达式可以确定值时，便不再进行运算右边表达式的值 一、逻辑与 的短路运算 逻辑与的表达式为 (表达式1 && 表达式2) 当表达式1为true时 则逻辑表达式的 返回值为 表达式2的值 当表达式1为false时 则逻辑表达式的返回值为表达式1的值
